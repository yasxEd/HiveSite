## Installation

### Prerequisites

- Node.js (for SASS compilation)
- npm (Node package manager)

### Steps

1. **Clone the Repository**
    ```sh
    git clone
    cd 
    ```

2. **Install Dependencies**
    ```sh
    npm install
    ```

3. **Compile SASS**
    ```sh
    npm run sass
    ```

4. **Open the Website**
    Open `index.html` in your web browser.
